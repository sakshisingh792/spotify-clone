## Spotify Clone

A Spotify-inspired music player UI built using HTML and CSS.
This project replicates the basic layout and design of the Spotify web interface to practice frontend development and responsive UI design.

## Features

Spotify-like user interface

Sidebar navigation menu

Music playlist layout

Album and song cards

Responsive design using CSS

Clean and modern UI

Tech Stack
